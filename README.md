# UniStock - Inventory Manager

Sistema de gerenciamento de estoque desenvolvido para controle de produtos, movimentação de estoque e análise de vendas em lojas de sneakers.

## Sobre o Projeto:

O UniStock é um sistema desenvolvido para auxiliar no gerenciamento de estoque de lojas de sneakers, permitindo controlar produtos, registrar movimentações de entrada e saída e gerar relatórios para apoio na tomada de decisão.

## Problema:

Muitas pequenas lojas utilizam planilhas ou métodos manuais para controlar o estoque, o que pode gerar erros, perda de produtos e dificuldade na análise de vendas.

## 💡 Solução

O sistema permite:

- Cadastro de produtos
- Controle de estoque
- Registro de entradas e saídas
- Relatórios de vendas
- Alertas de estoque baixo
